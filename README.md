# Lucas Alves Personal | Landing Page
 
Landing page desenvolvida para um personal trainer, com foco em conversão de visitantes em leads via WhatsApp. Projeto freelance, do protótipo em Figma à publicação em produção.
 
🔗 **Site no ar:** [lucasalvespersonal.com.br](https://lucasalvespersonal.com.br/)
 
## 📋 Sobre o projeto
 
O objetivo era criar uma página que apresentasse o trabalho do personal trainer Lucas Alves (CREF 067837-G/RJ), destacando resultados de alunos e depoimentos, com chamadas diretas para contato via WhatsApp.
 
O projeto começou com um protótipo no Figma, validado com o cliente, e evoluiu ao longo do desenvolvimento conforme feedbacks e novas necessidades foram surgindo.
 
## ✨ Funcionalidades
 
- Carrossel duplo (resultados e depoimentos) com comportamento adaptado por dispositivo:
  - Desktop: loop infinito suave via clonagem de elementos
  - Mobile: scroll nativo com scroll-snap
- Animações de scroll (fade/slide-in) usando IntersectionObserver
- Indicadores (dots) dinâmicos que sincronizam com o slide visível
- Design 100% responsivo, com tipografia e espaçamentos fluidos via `clamp()`
- CTAs diretos para WhatsApp em múltiplos pontos da página, com mensagens pré-definidas
## 🛠️ Tecnologias utilizadas
 
- HTML5 semântico
- CSS3 — variáveis customizadas, `clamp()` para responsividade fluida, metodologia BEM para nomenclatura de classes
- JavaScript — manipulação de DOM, IntersectionObserver, lógica de carrossel reutilizável e configurável
- Git para versionamento, com commits ao longo do desenvolvimento
- Netlify para deploy e hospedagem
